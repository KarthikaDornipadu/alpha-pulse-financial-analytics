Finance Concepts Used in AlphaPulse Project
1. Financial Markets and Stocks
Financial markets are platforms where securities like stocks are traded.
A stock represents ownership in a company such as Apple Inc., Tesla Inc., and Alphabet Inc..
In this project, historical stock price data is used to analyze performance and risk.
2. Stock Returns
Stock return measures the percentage change in price over time.
It helps evaluate profitability of an investment.
In this project:
Daily returns are calculated from closing prices
Used as the base for all further analysis (volatility, correlation, etc.)
4. Portfolio Concept
A portfolio is a combination of multiple stocks.
Instead of analyzing one stock:
This project combines multiple assets into a portfolio
Provides a more realistic investment scenario
Portfolio return depends on weights assigned to each stock
5. Risk and Volatility
Risk refers to uncertainty in returns.
Volatility is the statistical measure of that risk:
High volatility → Large price fluctuations
Low volatility → Stable prices
In this project:
Volatility is calculated using standard deviation of returns
Helps identify risky vs stable stocks
6. Rolling Volatility (Time-Based Risk)
Rolling volatility measures how risk changes over time.
In this project:
A 30-day rolling window is used
Shows periods of high and low market instability
Important for understanding dynamic market behavior
7. Correlation Between Stocks
Correlation measures how two stocks move relative to each other.
+1 → Move together
0 → No relationship
-1 → Move in opposite directions
In this project:
Correlation matrix is visualized using a heatmap in Power BI
Helps in diversification decisions
8. Diversification
Diversification means spreading investments across different assets.
Purpose:
Reduce overall portfolio risk
In this project:
Multiple stocks are combined
Correlation analysis supports diversification strategy
9. Monte Carlo Simulation
Monte Carlo simulation is a statistical method used to forecast future price movements.
In this project:
Random simulations are generated based on historical returns
Multiple possible future scenarios are created
Output:
Different simulated price paths
Final simulated values
Helps answer:
“What are the possible future outcomes of this portfolio?”
11. Risk-Return Tradeoff
A fundamental concept in finance:
Higher return → Higher risk
Lower risk → Lower return
In this project:
High-growth stocks show higher volatility
Stable stocks show lower risk
12. Key Performance Indicators (KPIs)
KPIs summarize financial performance.
In this project, key KPIs include:
Average Return
Portfolio Return
Volatility
Correlation values
Final Monte Carlo simulation value
13. Data-Driven Decision Making
Modern finance relies on data analytics.
This project demonstrates:
Use of Python/SQL for calculations
Use of Power BI for visualization
Insights derived from financial metrics

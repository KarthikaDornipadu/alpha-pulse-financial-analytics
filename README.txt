# AlphaPulse - Investment Risk & Volatility Monitor

## 📌 Project Overview

AlphaPulse is a financial analytics project designed to monitor investment risk and market volatility. It helps analyze stock performance, measure risk exposure, and support better portfolio decisions.

---

## 🚀 Features

* Stock price and return analysis
* Value at Risk (VaR) calculation
* Monte Carlo simulation (10,000 runs)
* Correlation heatmap between stocks
* Rolling volatility (30-day)
* Interactive Tableau dashboard

---

## 🛠️ Tech Stack

* Python
* NumPy
* yfinance API
* SQL
* Tableau

---

## 📊 Project Workflow

1. Collect stock data using yfinance
2. Clean and preprocess the data
3. Calculate daily returns
4. Perform risk and volatility analysis
5. Run Monte Carlo simulation
6. Visualize results in Tableau

---

## 📁 Folder Structure

* Data → Raw and processed datasets
* python → Analysis and simulation scripts
* SQL → Database queries
* Dashboard → Tableau files
* Documentation → Project reports

---

## 💡 Use Case

This project helps investors and analysts understand portfolio risk, identify correlations between assets, and make data-driven investment decisions.
